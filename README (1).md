# Sri Venkateswara Devasthanam — Demo Website

A personal demo project: a temple darshan-booking and seva-offering website with a
welcome photo banner. Not affiliated with or an official website of any real
temple trust.

## Files
- `index.html` — page structure
- `style.css` — maroon & gold temple-themed styling
- `script.js` — booking + donation form logic (saved in your browser)
- `images/welcome-photo.jpg` — your welcome banner photo

## Run it locally
Double-click `index.html` to open it in your browser.

## Deploy it (GitHub Pages)
Same steps as your to-do app:
1. Create a new GitHub repository
2. Upload all files **and the `images` folder** (drag the whole folder in)
3. Settings → Pages → Branch: main, folder: / (root) → Save
4. Visit the link GitHub gives you
